# mabandit

A python based library which includes multi arm bandit and Bayesian optimization algorithms.

## Installation
<code>pip install mabandit==1.3</code>

## Requirements
* Python 3
* Numpy
* Matplotlib
* tensorflow

## Features
* Beta bernoulli bandits
* Gaussian process bandits
* Baysian bandits
